# MongoDB Replica Set Runbook (Docker)

Bu repository, Docker üzerinde çalışan **MongoDB Standalone → Replica Set**
geçişi için hazırlanmış kapsamlı bir runbook içerir.

## İçerik

- `docs/README.md` – Mimari, kavramlar ve kurulum adımları
- `docs/TROUBLESHOOTING.md` – Sık karşılaşılan hatalar ve çözümleri
- `docs/ROLLBACK.md` – Geri dönüş (rollback) stratejileri
- `images/` – Mimari diyagramlar
- `CHANGELOG.md` – Doküman değişiklik geçmişi
- `LICENSE` – Lisans bilgisi

Bu doküman üretim ortamları (production) hedeflenerek hazırlanmıştır.
